# Olympics-World_Championships-Project

In this project i used HTML, CSS and JavaScript languages to bring this project to life.
The data used in this project was obtained via the Totallympics forum, Wikipedia and the websites related to the respective competitions.

## What's in it

For now, you can view the world championship medallists for the current summer olympic cycle (2024-2028) + you can view all the world championship events for the past summer + winter olympics cycles (2016-2021 and 2021-2024) and (2018-2022 and 2022-2026)

You can also view the events on past Summer and Winter Olympics and even the events of the future 2026 Winter Olympics in Milano-Cortina.

## What to do next

Keep building up the data for past Olympics and World Championships

## How to run

For now, using Visual Studio Code with the Live Server extension and clicking on it, in the bottom right corner of your screen you will open the current files on your browser. Select the HTML file you want and see how your favorite nation fared.
